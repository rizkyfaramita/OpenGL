# OpenGL Mouse Programming

## Usage

Clone the repository, or download the zip file of the release you would like to use. The source code is available in src folder. 

To compile, use:
```bash
gcc -o mouse mouse.c -lX11 -lGL -lGLU
```

To run, use:
```bash
./mouse
```
